Para llevar a cabo tus prompts debes seguir los siguientes pasos y te puedes apoyar de los tips mencionados.

Pasos para elaborar prompts.

    Paso No.1: Delimitar el prompt entre comillas ("") o cualquier signo que lo separe de las instrucciones.

    Paso No.2: Pedir que el output esté lo más estructurado posible, para que este sea fácil de leer y de comprender.

    Paso No.3: Pedir al modelo si cumplen todas las condiciones. esto para saber si entendió lo solicitado y desglosó la respuesta correcta.

    Paso No.4: Ofrecer un ejemplo del resultado esperado.

    Tip para el paso 4. Al pedir esto, ayuda al modelo a entender qué es lo que esperas de la respuesta, ya que ha tareas que pueden tener múltiples respuestas o que dichas respuestas se puedes obtener en diferentes formatos.

Principios fundamentales.

    Paso No.1: Especificar los pasos para terminar una tarea

    Paso No.2: Pedirle al modelo que trabaje en su propia solución; es importante que el modelo desarrolle un procedimiento antes de darte una respuesta (en ámbito matemático o algo que involucre operaciones) para que de una respuesta acertada.

TIPS.

Sigue estos tres tips para mejorar tus prompts...

Retroalimentación: Es decir, evaluar los resultados y modificar el prompt para obtener los resultados.

Resumir: Se puede resumir, compendiar y/o crear un extracto, cada uno dará un resultado diferente.

Inferencia: Proceso en el que se utiliza la información disponible para llegar a una conclusión o deducción.
